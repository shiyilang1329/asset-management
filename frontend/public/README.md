# 登录背景图片和Logo说明

## 需要添加的文件

### 1. 登录背景图片（可选）
- **文件名**：`login-bg.jpg`
- **建议尺寸**：1920x1080 或更高
- **位置**：`frontend/public/login-bg.jpg`
- **说明**：如果不添加，将使用默认的深蓝灰渐变背景

### 2. Logo图片（可选）
- **文件名**：`logo.png`
- **建议尺寸**：200x200 或更高（正方形）
- **位置**：`frontend/public/logo.png`
- **格式**：PNG（支持透明背景）
- **说明**：如果不添加，侧边栏和登录页将只显示文字标题

## 支持的图片格式

- JPG/JPEG（背景图推荐）
- PNG（Logo推荐，支持透明背景）
- WebP

如果使用其他格式或文件名，需要修改相应的代码文件。

## 文件结构示例

```
frontend/
  public/
    login-bg.jpg  ← 背景图片（可选）
    logo.png      ← Logo图片（可选）
```

## 效果预览

### 有Logo时
- 登录页面：显示Logo + 系统标题
- 侧边栏：显示Logo + 系统名称

### 无Logo时
- 登录页面：只显示系统标题
- 侧边栏：只显示系统名称

## 快速开始

1. 准备图片（可选）：
   - 背景图（横向，1920x1080）
   - Logo（正方形，200x200，PNG格式）

2. 重命名图片：
   - 背景图 → `login-bg.jpg`
   - Logo → `logo.png`

3. 将图片复制到 `frontend/public/` 文件夹

4. 刷新浏览器查看效果

## 注意事项

- Logo和背景图都是可选的，不添加也不影响系统使用
- 如果不添加Logo，系统会自动隐藏Logo区域
- 如果不添加背景图，登录页会使用默认的渐变背景
- 建议Logo使用透明背景的PNG格式，效果更好
